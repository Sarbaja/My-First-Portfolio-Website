a. What is a markup?
Ans: Markup are the language used for structing documents such as web browsers or screen readers.

b. What is hypertext and hypermedia?
Ans: Hypertext are those which links to other text.
Hypermedia are text which contains links to video,sound etc.

c. What is a doctype and why is it important to include it in every HTML document?
Ans: Doctype is an instruction written which identify the version of HTML used. It is very important to include as it defines which version of markup language is written in the text document. For example <header> </header> is a markup language for HTML5.

d. What language can be used to style web documents?
Ans: Casacading Style Sheets (CSS) is used to style web documents.

e. Which elements are required for a valid XHTML5 document? Ans: The elements required are Doctype, html, head, body, title.

f. What is a self closing HTML element? Give an example? Ans:Self closing HTML tag are those tag which needs to have a start tag but doesn't require to be closed and are self closed.For example <br> , <hr> , <meta> etc.

g. What is XML and why is it important to web developers? Ans: XML is an eXtensible Markup Language with stricter syntax rules than HTML. It is important to developers because it is essential for well structured documents that are easier to develop and maintain.

h. Describe 4 XML rules?
Ans: The 4 XML rules are
1. Elements must be properly nested.
2. Elements must have a start and end tag.
3. Elements names are case-sensitive.
4.Attributes may not be repeated.

h. What is the difference bwtween a "well-formed" and a valid web document?
Ans: Well formed are the documents adhering to XML and are ideal for manipulation by computer programs such as browsers whereas a document that conforms to a particular doctype and is also "well-formed" is considered a valid document.
