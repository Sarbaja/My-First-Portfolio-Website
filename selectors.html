Type selectors are those selector which gives style to the elements we select. For example p{color:blue;}

class selectors are those selector which has a (.) in beginning of a class name and also an ID selector. For example .red{color:red}

ID selector is a name preceded by a hash character.

pseudo-class is a keyword added to selectors that specifies a special state of the element to be selected. For example: a:visited

combination is a selector can contain more than one simple selector.
